## Colors
Colors are used to specify text colors mainly. We can also use these color values to set background colors to specific elements.

We can specify colors with the following methods:
- Predefined color names
- HEX value
- RGB value
- RGBA value
- HSL value
- HSLA value

# CSS
CSS stands for Cascading Style Sheets.

We can use CSS in our HTML documents with various methods such as:
- Inline CSS
- Internal CSS
- External CSS

## Links
- HTML links are called Hyperlinks.
- We can click on a link and jump to another document.
- When we move the mouse over a link the arrow will turn into a little hand.

#### Target Attributes
- It specifies where to open the linked document.

- The target attribute can have one of the following values:
    - _self: This is the default value which helps to open the document in same window or tab as it was clicked.
    - _blank: It helps to open the linked document in a new window or tab.
    - _parent: It opens the linked document in the parent frame.
    - _top: It opens the linked document in the full body of the window.

## images
We include images in our web page using ```<img>``` tag.

```<img>``` has various attributes:
    - src: We define source of the image either in absolute url or relative url.
    - width: We define width of the image.
    - height: We define height of the image.
    - alt: We call it alternative text. It is being displayed to the user only when image failed to load.
